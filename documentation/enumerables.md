# Enumerables

Here is a list of enumerables that I use pretty frequently.
[Read more at Ruby-Docs](http://ruby-doc.org/core-2.2.3/Enumerable.html)

### Information
- `#count`
- `#first`
- `#include?`
- `#none?`
- `#max | #min`
- `#max_by | #min_by`
- `#ceil | #floor`

### Transformation
- `#collect / #map`
- `#detect / #find`
- `#select`
- `#inject / #reduce`
- `#reject`, `#sort`
- `#sort_by`
- `#to_a`
- `#group_by`

### Iteration
- `#each_with_index`
- `#reverse_each`
- `#while | #until`
- `#case`
- `#times`
