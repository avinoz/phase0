Ruby v.s. Javascript

08/28/15

A computer is an electronic machine that takes raw data, stores it until it’s ready to do it and then returns another set of data. In technical terms it is an information processor that takes input data, stores it in memory, processes it and output the results.

As a means of input we have things such as a keyboard, a mouse or a camera etc to create data. Next we have memory cards or hard drives which store all the information and the CPU (central processing unit) microchip that does the hard work and then outputs data such as characters or an image on your LCD screen.

So all that we’ve talked about above is hardware. It’s the physical objects that makes it work. Software on the other hand can be thought of as reprogramming the hardware parts to do things you’d want it to like adding numbers or changing the contrast level of an image. This type of software would be called applications.

There are four levels to a computer from the ground up: hardware, bios, operating system and applications. We know what the first and the last does but what is an operating system? Well, it’s the core software of a computer that controls the basic functions such as the input, output, storing and processing. Bios (Basic Input Output System) is a more fundamental version of the operating system linking the hardware to the software.
